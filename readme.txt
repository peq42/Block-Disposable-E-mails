A WordPress Plugin for Blocking Disposable Burner E-mails.
 
== Description ==
 
Annoyed with disposable e-mails being used to register to your website? Tired of newsletters being sent to burner e-mails? With this free open-source plugin, those issues are gone! 

Very simple to use, optimized for performance, open-source code, bad e-mail list constantly updated.
 
 
== Frequently Asked Questions ==
 
= How do I use this plugin? =
 
This plugin is plug-and-play. Soon after installed, it'll block people from registering using burner e-mails.
 
= How do I use it on other things? =
 
If you wish to use this plugin on other parts of your website, like newsletters, comments, etc, you can do so by calling its function "email_check($errors, $sanitized_user_login, $user_email)" on the e-mail and checking if any errors were returned.
